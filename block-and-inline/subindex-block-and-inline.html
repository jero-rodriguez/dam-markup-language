Este apartado es el que nos muestra los elementos Block y Line

Una explicación de por qué alunos elementos se comportan de manera distinta

Un elemento de nivel de bloque siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y hacia la derecha tanto como sea posible). (<div> ).

Un elemento en línea no comienza en una nueva línea y solo ocupa el ancho que sea necesario. ( ej.<span>,<a>,<img>). Este parrafo tiene borde, podemos apreciar su tamaño.

Esto es un ejemplo de div element, tambien con borde, para que veamos lo que ocupa

Los elementos <h1>, <h2>, <div> y <o>. Son ejemplos de elementos de bloque. Fuerzan a que por defecto esten uno debajo de otro.

Los elementos como <image>. Son ejemplos de elementos de elemento en linea. Fuerzan a que por defecto esten uno debajo de otro.

En el caso de que queramos que un elemento en línea se comporte como un elemento en bloque, tenemos que meterlo dentro de un elemento de bloque. Marcare el borde para que se vea claramente.
