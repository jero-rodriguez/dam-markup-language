El atributo style sirve para dar formato CSS a una etiqueta de HTML

La forma general de un estilo es: 
< etiqueta style="property:value; another-property: another-value">

Algunos ejemplos de formateo de etiqueta:

  • <body style="background-color:powderblue;"></body>
  • <h1 style="color:blue;">Cabecera en azul</h1">
  • <p style="font-family:courier;"">This is a paragraph.</p">

Algunos ejemplos de formateo de texto:

<b> - Texto en negrita. Esta etiqueta no debería usarse, hay que emplear CSS
<strong> - Texto importante. Se ve igual que negrita, pero aporta sentido semántico.
<i> - Texto en cursiva. Esta etiqueta no debería usarse, hay que emplear CSS
<em> - Texto enfatizado. Mismo caso que la etiqueta <strong>
<mark> - Texto marcado
<small> - Texto pequeño
<del> - Texto eliminado
<ins> - Texto insertado
<sub> - Texto del <sub>
<sup> - Texto superíndice
