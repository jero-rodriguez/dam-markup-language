Enlaces en HTML

En este capitulo vamos a conocer los enlaces y como funcionan.

Enlace

La etiqueta de enlace es la siguiente:

<a href="https://www.iesjuanbosco.es/" target="_blank">Visita nuestro Instituto</a>

Aqui podemos identificar los atributos siguientes

  • href: Que nos indica a donde apunta el enlace
  • target

El atributo de target puede tener uno de los siguientes valores:

  • _blank
    Abre el documento vinculado en una nueva ventana o pestaña

    Long Doge Challenge

  • _self
    Abre el documento vinculado en la misma ventana / pestaña en la que se hizo clic (esto es predeterminado)

    Memory Game

  • _parent
    Abre el documento vinculado en el marco primario

    Puzzle

  • _top
    Abre el documento vinculado en el cuerpo completo de la ventana

    Caligrama
